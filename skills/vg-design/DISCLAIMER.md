### Relationship to Sources

- **emil-design-eng** (Emil Kowalski) is the sole authoritative standard for all motion, easing, and animation decisions within `vg-design`. Where other sources conflict with Emil's standards, Emil wins.
- **vercel-composition-patterns** (Vercel) provides the React architecture and component API philosophy.
- **impeccable**, **design-taste-frontend**, and **high-end-visual-design** (Matt Pocock) provide the broader design system, anti-slop guardrails, premium UI patterns, and quality enforcement.
- **full-output-enforcement** (Matt Pocock) provides the output quality baseline.
